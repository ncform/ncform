ncform，一种令人愉悦的表单开发方式，仅需配置即可生成表单UI及其交互行为。

自带丰富的 **标准组件** 和 **校验规则**，开箱即用。

具备强大的 **控件交互** 和 **扩展能力**，做你所想。
